## Installation

NNNetwork is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'NNNetwork'
```

>  该封装只是对AFNetworking的二次封装，在实际项目使用时，请再进行二次封装，如单例等。
